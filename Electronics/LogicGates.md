# Electronics - Logic Gates
---
![Logic Gates](assets/logic_gates.jpg)

**Buffer Gate:** Outputs the same value received for the input.
| Input   | Output  |
| :-----: | :-----: | 
|    1    |    1    |
|    0    |    0    |

**NOT Gate:** Outputs the complementary of a, or: $\overline{\rm a}$

| Input   | Output  |
| :-----: | :-----: | 
|    1    |    0    |
|    0    |    1    |
